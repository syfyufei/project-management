# Project Management

Comprehensive project management skill for creating, restructuring, validating, and monitoring research and analysis projects

## Installation

In Claude Code, register the marketplace first:

```bash
/plugin marketplace add syfyufei/project-management
```

Then install the plugin from this marketplace:

```bash
/plugin install project-management@project-management-marketplace
```

### Verify Installation

Check that commands appear:

```bash
/help
```
