# Validate

This command validates project structure against standards and provides detailed scoring.

Parameters:
- root (optional): Project directory to validate (default: current)
- fix_issues (optional): Automatically fix structural issues (default: false)
- strict_mode (optional): Enable strict validation (default: false)

The command will:
1. Check required directory structure
2. Validate required files exist and are properly formatted
3. Evaluate project structure completeness
4. Calculate compliance score (0-100)
5. Provide detailed fix suggestions
6. Auto-fix issues if enabled
7. Generate comprehensive validation report
