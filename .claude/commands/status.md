# Status

This command reports detailed project status and comprehensive statistics.

Parameters:
- root (optional): Project directory to analyze (default: current)
- include_git (optional): Include Git statistics (default: true)
- include_file_stats (optional): Include detailed file statistics (default: true)

The command will:
1. Scan project structure and generate metrics
2. Count files by type and directory
3. Analyze project metadata from configuration files
4. Track file sizes and modification dates
5. Provide Git statistics if enabled
6. Generate visual status indicators
7. Return comprehensive status report
